B.E Computer Science and Engineering (CSE)

The B.E. Computer Science and Engineering program is designed to provide students with strong foundations in programming, software development, networking, databases, and emerging technologies.

Subjects

  • Python Programming
  • Java Programming
  • Database Management System (DBMS)
  • Operating Systems (OS)
  • Computer Networks
  • Data Structures

Course Details

Duration
4 Years (8 Semesters)
Eligibility
12th Pass with Mathematics and Physics
Fees
₹75,000 per year
Degree Awarded
Bachelor of Engineering (B.E.)

Learning Path

  1. Learn Programming Fundamentals
  2. Understand Data Structures
  3. Study Database Management Systems
  4. Learn Operating Systems
  5. Develop Web Applications
  6. Complete Final Year Project
